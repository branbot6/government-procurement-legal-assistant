## 什么情况下可以采用邀请招标？

**1、结论**

依据现有证据，**未找到关于邀请招标具体适用条件的直接规定**。证据1（财政部令第87号第十四条）仅规定了邀请招标的程序性操作方式，未明确列举适用情形。

**2、核心要求清单**

根据证据1，采用邀请招标时必须遵守以下程序要求：

| 环节 | 要求 |
|------|------|
| 供应商产生方式 | 通过以下方式之一产生：①发布资格预审公告征集；②从省级以上财政部门建立的供应商库中选取；③采购人书面推荐 |
| 供应商数量 | 随机抽取3家以上供应商 |
| 备选要求 | 采用第二、三项方式的，备选供应商总数不得少于拟随机抽取供应商总数的两倍 |
| 随机抽取规则 | 通过抽签等方式保证机会均等，不少于两名采购人工作人员在场监督，形成书面记录存档 |
| 投标邀请 | 应同时向所有受邀请供应商发出 |

**3、依据条款**

- 条款：第十四条
  - 来源法规：政府采购货物和服务招标投标管理办法(中华人民共和国财政部令第87号)

- 条款：第四条
  - 来源法规：财政部关于印发《政府和社会资本合作项目政府采购管理办法》的通知（财库〔2014〕215号）

- 条款：第二十七条
  - 来源法规：政府采购非招标采购方式管理办法(中华人民共和国财政部令第74号)

**4、适用条件**

现有证据未明确列出邀请招标的适用情形。根据证据2，PPP项目可依法选择邀请招标方式；证据4提及公开招标供应商不足两家时可转为其他方式，但均未直接界定邀请招标的适用条件。

**5、风险提示**

❓ **依据不足**：关于邀请招标的适用情形（如技术复杂、供应商特定等），在提供的证据中未找到明确规定。建议查阅《政府采购法》第二十九条及《招标投标法实施条例》第八条等上位法规获取完整依据。
