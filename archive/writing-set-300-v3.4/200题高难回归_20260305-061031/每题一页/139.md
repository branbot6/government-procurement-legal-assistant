# 第 139 题（定义精度）

## 问题
请给出采购方式变更在国家层面的法律定义、构成要件和排除要件，并指出最容易被误用的口径。

## Top1 证据标题
中华人民共和国财政部令第74号《政府采购非招标采购方式管理办法》 第二条

## 回答
**一、结论**

1. **法律定义**：在国家层面，"采购方式变更"并非一个法定的独立概念，而是实践中对采购方式调整的通称。从现行法规体系看，主要涉及**非招标采购方式**（财政部令第74号）与**框架协议采购方式**（财政部令第110号）两大类别。

2. **构成要件**：采购方式的适用需满足法定条件。非招标采购方式适用于竞争性谈判、单一来源采购、询价三种情形；框架协议采购需符合财政部令第110号第三条规定的四种特定情形。

3. **排除要件**：证据4、5、6明确指出，属于第三条第一款第二项情形下，若主管预算单位能够归集需求形成单一项目进行采购并通过签订时间、地点、数量不确定的合同满足需求的，**不得采用框架协议采购方式**。

4. **最容易被误用的口径**：框架协议采购方式的适用条件最容易被误用，特别是对"小额零星采购"的界定以及对"归集需求形成单一项目"的排除条件把握不准。

---

**二、依据条款**

1. **《政府采购非招标采购方式管理办法》（财政部令第74号）第二条**——明确了非招标采购方式的概念和三种类型（竞争性谈判、单一来源采购、询价）。

2. **《政府采购框架协议采购方式管理暂行办法》（财政部令第110号）第三条第一款**——规定了框架协议采购方式的四种适用情形及排除条件。

---

**三、适用条件**

| 采购方式类别 | 适用条件 |
|-------------|----------|
| **竞争性谈判** | 谈判小组与符合资格条件的供应商就采购货物、工程和服务事宜进行谈判，供应商提交响应文件和最后报价 |
| **单一来源采购** | 采购人从某一特定供应商处采购货物、工程和服务 |
| **询价** | 询价小组向符合资格供应商发出询价通知书，供应商一次报出不得更改的价格 |
| **框架协议采购** | ①集中采购目录以内品目及配套耗材配件，属于小额零星采购；②集中采购目录以外、限额标准以上，本部门行政管理所需的鉴证咨询服务，属于小额零星采购；③集中采购目录以外、限额标准以上，为本部门以外服务对象提供服务的政府购买服务项目；④国务院财政部门规定的其他情形 |

---

**四、不确定点**

1. **"采购方式变更"缺乏明确的法律定义**：现行法规体系中并无"采购方式变更"的专门定义或条款，实践中对"变更"的理解可能存在分歧。

2. **"小额零星采购"的界定标准**：财政部令第110号虽使用"小额零星采购"概念，但未明确具体数额标准或认定标准。

3. **采购方式之间的转换规则**：证据中未涉及从一种采购方式转换为另一种采购方式的条件和程序规定。
