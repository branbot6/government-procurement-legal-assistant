# 第 033 题（证据充分性）

## 问题
若仅有会议纪要、邮件、评分表三类材料，是否足以证明价格分异常中的禁止性要求合法？还缺哪些关键证据？

## Top1 证据标题
政府采购信息公告管理办法 第二十三条

## 回答
## 结论

仅凭会议纪要、邮件、评分表三类材料，**不足以**充分证明价格分异常中禁止性要求合法。缺少多项关键证据材料。

## 核心要求清单

证明价格分合法需要提供以下证据链：
1. 采购文件（招标文件/磋商文件）——证明评审因素已在文件中明确规定
2. 评审方法说明——证明采用综合评分法符合规定
3. 价格分计算过程——证明价格分计算符合"低价优先法"
4. 政策执行依据——证明中小企业价格扣除等政策已落实

## 依据条款

- **条款**：第五十五条；**来源法规**：财政部令第87号《政府采购货物和服务招标投标管理办法》
- **条款**：第十二条；**来源法规**：政府采购促进中小企业发展管理办法
- **条款**：第二十四条；**来源法规**：财政部关于印发《政府采购竞争性磋商采购方式管理暂行办法》的通知（财库〔2014〕214号）

## 适用条件

综合评分法价格分合法的条件：
- 货物项目价格分值占总分值比重≥30%
- 服务项目价格分值占总分值比重≥10%
- 采用"低价优先法"计算价格分
- 评审因素已在招标文件中明确规定并量化
- 中小企业价格扣除比例在法定幅度内（6%-10%）

## 风险提示

现有三类材料的证明力有限：
- **会议纪要**：仅能证明内部讨论过程，不能替代法定的采购文件
- **邮件**：通常为沟通记录，缺乏正式文件效力
- **评分表**：仅体现评分结果，无法反映评分方法合规性

若发生质疑投诉，无法仅凭上述材料证明价格分设置的合法性。
