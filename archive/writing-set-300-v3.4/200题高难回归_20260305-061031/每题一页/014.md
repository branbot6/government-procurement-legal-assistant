# 第 014 题（复合场景）

## 问题
同一项目先后发生履约验收与投诉处理并行，程序交叉时如何避免“程序互相污染”？请给最小风险流程图式文字说明。

## Top1 证据标题
中华人民共和国财政部令第87号《政府采购货物和服务招标投标管理办法》 第六条

## 回答
## 1) 结论

**履约验收与投诉处理并行时，应建立"程序隔离、证据独立、流程分栈"的内部控制机制，以投诉处理程序优先为原则，通过人员隔离、信息隔离和证据隔离避免程序污染。**

## 2) 核心要求清单

- **投诉处理优先**：任何涉及同一项目的投诉处理应优先于履约验收程序
- **人员隔离**：参与投诉处理的人员不得参与履约验收评审；验收组成员与投诉处理人员不得重叠
- **信息隔离**：履约验收材料与投诉处理材料分别归档、分别审批
- **证据独立**：验收记录与投诉调查材料分别形成独立证据链，不得交叉使用
- **流程分栈**：建立并行业务的"防火墙"，分别走独立审批流程

## 3) 依据条款

- **条款**：第六条（采购人应当按照行政事业单位内部控制规范要求，建立健全本单位政府采购内部控制制度，在编制政府采购预算和实施计划、确定采购需求、组织采购活动、履约验收、答复询问质疑、**配合投诉处理**及监督检查等重点环节加强内部控制管理）
  - **来源法规**：政府采购货物和服务招标投标管理办法(中华人民共和国财政部令第87号) 第六条

- **条款**：第十三条（参与论证的专家不得作为采购评审专家参与同一项目的采购评审工作）
  - **来源法规**：财政部关于印发《政府采购进口产品管理办法》的通知（财库[2007]119号） 第十三条

## 4) 适用条件

- 适用于同一政府采购项目在履约验收阶段同时收到供应商投诉或质疑的情形
- 适用于采购人已启动履约验收程序后，供应商提出投诉需要调查处理的情形
- 适用于项目金额较大、程序较复杂、涉及多个供应商权益的采购项目

## 5) 风险提示

- **程序混同风险**：若验收与投诉处理程序未隔离，可能导致验收结论因投诉因素产生偏差，或投诉处理因验收结果形成预断
- **证据污染风险**：验收材料与投诉调查材料混同可能导致证据链不清晰，影响后续行政复议或诉讼
- **人员利益冲突**：验收人员若参与投诉处理，可能影响投诉处理的公正性

## 最小风险流程图式文字说明

```
┌─────────────────┐
│  事项并行发生   │
└────────┬────────┘
         ▼
┌─────────────────────────────────┐
│  第一步：投诉处理程序优先启动   │
│  → 暂停或暂缓履约验收程序      │
└────────┬────────────────────────┘
         ▼
┌─────────────────────────────────┐
│  第二步：人员隔离组建立         │
│  → 成立投诉处理专项小组        │
│  → 验收组与投诉组零人员重叠    │
└────────┬────────────────────────┘
         ▼
┌─────────────────────────────────┐
│  第三步：材料分流归档           │
│  → 验收材料入"验收档案"        │
│  → 投诉材料入"投诉处理档案"    │
│  → 各自独立编号、分别审批      │
└────────┬────────────────────────┘
         ▼
┌─────────────────────────────────┐
│  第四步：程序分别推进           │
│  → 投诉处理作出最终决定后      │
│  → 履约验收再行启动或恢复      │
└────────┬────────────────────────┘
         ▼
┌─────────────────────────────────┐
│  第五步：结论互不溯及           │
│  → 验收结论依据验收证据独立形成 │
│  → 投诉处理结论依据调查证据独立形成 │
└─────────────────────────────────┘
```

**核心逻辑**：以"程序隔离"为防火墙，以"证据独立"为底线，确保投诉处理不影响验收客观性，验收结果不污染投诉处理公正性。
