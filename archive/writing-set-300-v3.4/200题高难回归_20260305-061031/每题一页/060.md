# 第 060 题（抽象原则落地）

## 问题
“公开透明、公平竞争、公正、诚实信用”在竞争性磋商中的可审计落地动作分别是什么？如何形成可留痕证据链？

## 结果状态
- status: OK
- evidence_count: 6
- top_score: 0.47470341287321005
- anchor: 公开透明

## Top1 证据标题
中华人民共和国财政部令第87号《政府采购货物和服务招标投标管理办法》 第二十五条

## 回答
## 结论

在竞争性磋商中，"公开透明、公平竞争、公正、诚实信用"四个原则可通过全流程标准化操作实现可审计落地，并通过档案留痕形成完整证据链。

## 核心要求清单

| 原则 | 落地动作 | 关键证据链 |
|------|----------|------------|
| **公开透明** | 1.磋商文件公示（不少于10日）<br>2.开标全程公开进行<br>3.中标结果公示<br>4.合同公告 | 公告发布记录、开标现场录像、公示截图、签到表 |
| **公平竞争** | 1.采购需求论证<br>2.磋商文件不含指向性条件<br>3.资格条件合理性审查<br>4.评审因素量化 | 论证意见书、文件审核记录、资格审查结果、评分表 |
| **公正** | 1.专家随机抽取<br>2.评审全程录音录像<br>3.回避制度执行<br>4.评分结果汇总确认 | 专家抽取记录、视听资料、回避声明、评审报告 |
| **诚实信用** | 1.响应文件真实性承诺<br>2.报价合理性说明<br>3.合同实质性条款遵守<br>4.履约验收按标准执行 | 承诺书、说明函、合同文本、验收报告 |

## 依据条款

1. **条款**：招标文件、资格预审文件的内容不得违反公开透明、公平竞争、公正和诚实信用原则
   - **来源法规**：政府采购货物和服务招标投标管理办法（财政部令第87号）第二十五条

2. **条款**：政府采购活动遵循公开透明、公平竞争、公正和诚实信用原则
   - **来源法规**：扬州市本级政府采购环节操作规范实施细则（扬财规〔2018〕2号）第三条

3. **条款**：竞争性磋商文件应当根据采购项目的特点和采购人的实际需求制定，并经采购人书面同意，采购人应当以满足实际需求为原则，不得擅自提高经费预算和资产配置等采购标准。磋商文件不得要求或者标明供应商名称或者特定货物的品牌，不得含有指向特定供应商的技术、服务等条件
   - **来源法规**：扬州市财政局关于印发扬州市政府采购采购人主体责任清单的通知（扬财购〔2022〕25号）第二十一条第六款

4. **条款**：采购人提供的采购需求书内容应符合法律、行政法规、强制性标准、政府采购政策要求，不得规定以注册资本、资产总额等规模条件作为资格要求或评审因素；不得设定与采购项目具体特点和实际需要不相适应的资格条件；采购需求不得指向特定供应商、特定产品
   - **来源法规**：扬州市本级政府采购环节操作规范实施细则（扬财规〔2018〕2号）第六条第二款

5. **条款**：评审因素应当细化和量化，且与相应的商务条件和采购需求对应
   - **来源法规**：扬州市本级政府采购环节操作规范实施细则（扬财规〔2018〕2号）第七条

6. **条款**：采用竞争性磋商方式采购的，自磋商文件发出之日起至供应商提交首次响应文件截止之日止，公告期不得少于10日
   - **来源法规**：扬州市本级政府采购环节操作规范实施细则（扬财规〔2018〕2号）第十七条

7. **条款**：采购人或者代理机构应当对开标、评标现场活动进行全程录音录像，录音录像应当清晰可辨，音像资料作为采购文件一并存档
   - **来源法规**：扬州市本级政府采购环节操作规范实施细则（扬财规〔2018〕2号）第二十条第九款

8. **条款**：采购项目档案包括预算执行文件、前期准备文件、开标文件、评审文件、中标（成交）文件、合同文件及开评标现场录音录像资料等，档案保存期限为从采购结束之日起至少保存15年
   - **来源法规**：扬州市本级政府采购环节操作规范实施细则（扬财规〔2018〕2号）第四十二条

## 适用条件

- 适用于扬州市市级政府采购目录以内或以外的竞争性磋商项目
- 采购金额达到竞争性磋商起点标准（需参照当年采购限额标准）
- 采购人委托代理机构组织实施的情形

## 风险提示

1. **程序瑕疵风险**：公告期限不足、文件公示不完整可能导致采购结果无效
2. **评审公正风险**：专家抽取不规范、评分标准未量化可能引发质疑投诉
3. **证据灭失风险**：未全程录音录像或档案保存不当，影响审计追溯
4. **合规性风险**：磋商文件含有指向性条件违反公平竞争，可能被责令整改
