财政部 监察部关于印发《集中采购机构监督考核管理办法》的通知 第二十三条

## 回答
## 结论

“形式合规但实质违规”是指采购文件的程序和形式符合法规要求，但采购需求设置存在实质性违法违规问题，常见于采购需求与合同条款不匹配、技术参数排他、资格条件歧视等情形。

## 核心要求清单

| 类别 | 要求 |
|------|------|
| 采购需求 | 完整、明确，符合法律法规及政府采购政策要求 |
| 合同文本 | 包含法定必备条款，与采购需求一致 |
| 资格审查 | 不得存在歧视性条件 |

## 依据条款

1. **条款**：采购需求应当符合法律法规以及政府采购政策规定的技术、服务、安全等要求；应当完整、明确
   **来源法规**：中华人民共和国政府采购法实施条例 第十五条

2. **条款**：合同权利义务要围绕采购需求和合同履行设置
   **来源法规**：政府采购需求管理办法 第二十三条

3. **条款**：采购代理机构应当提高确定采购需求，编制招标文件的专业化服务水平
   **来源法规**：中华人民共和国政府采购法实施条例 第十三条

## 适用条件

以下情形可认定为“形式合规但实质违规”：

- 采购需求技术参数仅指向特定供应商或产品
- 合同条款与采购需求内容不一致
- 资格预审条件设置与采购项目实际需求无关
- 评分标准与采购需求重点不匹配

## 风险提示

| 风险类型 | 可能后果 |
|----------|----------|
| 程序合法但需求违规 | 可能被认定为以不合理条件限制或排斥潜在供应商 |
| 需求与合同不一致 | 引发合同纠纷，影响项目验收和资金支付 |
| 审查不严 | 财政部门监督考核时可能追责 |

## 纠偏动作

1. **需求审查**：对照采购需求逐项核对合同条款，确保一致性
2. **技术评审**：组织专家论证技术参数是否存在唯一性指向
3. **合规性复核**：审查资格条件是否与项目实际需要相匹配
4. **整改程序**：发现问题的，及时修订采购需求文件并重新履行审批程序
