# 第 093 题（地方与国家层级）

## 问题
在扬州市办理质疑答复时，国家规范、江苏省规范、扬州市细则同时存在，适用顺序和冲突消解规则是什么？

## Top1 证据标题
江苏省财政厅关于印发《政府采购投诉处理流程》的通知（苏财购〔2018〕32号） 第六条

## 回答
1) 结论

在扬州市办理质疑答复时，应适用以下顺序：**法律、行政法规 → 财政部部门规章 → 江苏省规范性文件 → 扬州市实施细则**。当发生冲突时，上位法优于下位法；同一层级法规有新规定的，按新规定执行。

2) 核心要求清单

- 质疑答复主体：采购人负责；采购人委托代理机构的，代理机构在委托授权范围内作出答复（证据6第三十八条）
- 质疑答复期限：收到质疑函后7个工作日内（证据3第四十二条第二款）
- 质疑答复内容应包括：质疑供应商姓名/名称、收到质疑函日期、质疑项目名称及编号、质疑事项、具体答复内容、事实依据和法律依据、告知投诉权利、答复人名称、答复日期（证据2第十三条）
- 供应商对答复不满意或采购人/代理机构未在规定时间答复的，可在答复期满后15个工作日内向同级财政部门投诉（证据6第四十条）

3) 依据条款

- **条款**：第三十八条；来源法规：《扬州市本级政府采购环节操作规范实施细则》（扬财规〔2018〕2号）
- **条款**：第四十二条第二款；来源法规：《江苏省财政厅关于印发〈政府采购质疑处理流程〉的通知》（苏财购〔2018〕86号）
- **条款**：第十三条；来源法规：《江苏省财政厅关于印发〈政府采购质疑处理流程〉的通知》（苏财购〔2018〕86号）

4) 适用条件

- 供应商应为参与所质疑项目采购活动的供应商（潜在供应商亦可）
- 质疑须以书面形式在法定期限内提出
- 质疑事项须属于政府采购活动范畴

5) 风险提示

- 扬州市细则不得与上位法（法律、行政法规、部门规章）相抵触，否则无效
- 如江苏省规范与国家规范冲突，应优先适用国家规范（上位法优先原则）
