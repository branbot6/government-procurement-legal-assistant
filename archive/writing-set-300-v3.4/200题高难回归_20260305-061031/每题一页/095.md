# 第 095 题（抽象原则落地）

## 问题
“公开透明、公平竞争、公正、诚实信用”在采购需求编制中的可审计落地动作分别是什么？如何形成可留痕证据链？

## Top1 证据标题
中华人民共和国财政部令第87号《政府采购货物和服务招标投标管理办法》 第二十五条

## 回答
## 结论

采购需求编制中落实"公开透明、公平竞争、公正、诚实信用"四项原则，需在需求论证、文件编制、公告发布、评审机制等环节设置具体可审计动作，并形成完整留痕证据链。

## 核心要求清单

### 一、公开透明原则

**落地动作：**
1. 采购需求书须明确投标人资质、项目需求、采购预算、评审标准和付款方式
2. 采购文件须在财政部门指定的政府采购网站、公共资源交易网站公示
3. 采购公告期限须符合法定要求（公开招标≥20日、竞争性磋商≥10日等）
4. 开标评标全程录音录像，音像资料作为采购文件一并存档

**证据链：** 采购需求书→采购公告截图→采购文件公示记录→开标记录及录音录像

### 二、公平竞争原则

**落地动作：**
1. 采购需求不得含有限制或排斥潜在供应商的条件（禁止8类情形）
2. 资格条件不得作为评审因素，评审因素须与商务条件和采购需求对应
3. 技术、服务要求不得指向特定供应商、特定产品或特定品牌
4. 采购需求须经专家论证，出具书面论证意见

**证据链：** 需求论证专家意见书→采购文件（含公平竞争性自查表）→论证小组书面意见

### 三、公正原则

**落地动作：**
1. 设定与货物服务质量相关的评审标准（投标报价、技术水平、履约能力、售后服务）
2. 评审因素须细化量化，与采购需求指标对应
3. 评审专家须从省级专家库随机抽取，名单在确定中标人前严格保密
4. 评标委员会成员须包含技术和经济专家（技术经济专家≥成员总数2/3）

**证据链：** 评审专家抽取记录→评标委员会组建文件→评分细则→评标报告

### 四、诚实信用原则

**落地动作：**
1. 采购需求内容须符合法律、行政法规、强制性标准、政府采购政策
2. 不得擅自提高经费预算和资产配置等采购标准
3. 合同签订不得对招标文件确定事项作实质性修改
4. 采购人须按合同约定及时支付采购资金

**证据链：** 采购需求合规性自查承诺→政府采购合同→验收报告→付款凭证

## 依据条款

- **条款：** 第二十五条  
  **来源法规：** 政府采购货物和服务招标投标管理办法(中华人民共和国财政部令第87号)

- **条款：** 第三条  
  **来源法规：** 扬州市本级政府采购环节操作规范实施细则（扬财规〔2018〕2号）

- **条款：** 第二十一条（一）  
  **来源法规：** 扬州市政府采购采购人主体责任清单（扬财购〔2022〕25号）

## 适用条件

适用于采购人编制采购需求、代理机构编制采购文件的全部政府采购项目，重点适用于：
- 采购金额较大的项目
- 技术复杂难以把握的项目
- 潜在供应商对采购需求有质疑的项目

## 风险提示

1. **合规风险：** 采购需求含有限制竞争情形可能导致废标或中标结果无效
2. **审计风险：** 缺少论证记录、公示不完整可能导致审计不合格
3. **投诉风险：** 违反公平竞争原则易引发供应商质疑投诉
4. **履约风险：** 未按合同约定履行义务需承担违约责任
