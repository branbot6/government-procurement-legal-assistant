# 第 074 题（合规整改）

## 问题
若历史项目在合同变更中已形成程序性违法但合同已履行过半，如何在不扩大损失前提下做合规整改与责任切分？

## 结果状态
- status: OK
- evidence_count: 6
- top_score: 0.24326235293179832
- anchor: 整改

## Top1 证据标题
中华人民共和国政府采购法  第七十三条 （一）

## 回答
1) 结论
合同已履行过半（属于“合同已履行”情形）下，程序性违法不影响合同效力，但需进行合规整改并由责任人承担赔偿责任。合规整改应立即停止违规变更行为、修正后续履行程序；责任切分应区分采购人责任与供应商责任，由过错方承担损失。

2) 核心要求清单
- **立即纠正**：停止违规的合同变更行为，按原合同约定继续履行
- **损失评估**：核算已履行部分因程序违法造成的额外成本或损失
- **责任认定**：根据违规变更的发起方确定责任人（采购人违规则采购人担责，供应商违规则供应商担责）
- **赔偿处理**：由责任人向受损方承担赔偿责任
- **后续合规**：完善内部审批流程，确保后续履行程序合法

3) 依据条款
- 条款：第七十三条第三项；来源法规：中华人民共和国政府采购法
- 条款：第七十二条第四项；来源法规：中华人民共和国政府采购法实施条例
- 条款：第三十一条第一款第四项；来源法规：中华人民共和国财政部令第94号《政府采购质疑和投诉办法》
- 条款：第三十二条第一款第四项；来源法规：中华人民共和国财政部令第94号《政府采购质疑和投诉办法》
- 条款：第五十五条第三项；来源法规：中华人民共和国财政部令第74号《政府采购非招标采购方式管理办法》

4) 适用条件
- 适用于政府采购合同已经履行（即合同已签订且开始执行）的情形
- 适用于程序性违法（如变更未经过法定审批程序、变更未依法公告等）影响中标/成交结果的情况
- 需满足“给采购人、供应商造成损失”的前提，方涉及赔偿责任

5) 风险提示
- 程序性违法不必然导致合同无效，但可能导致行政责任（如责令改正、警告、罚款）
- 若程序性违法严重到影响中标/成交结果有效性，可能被认定为成交无效，需重新开展采购活动
- 责任切分时应固定证据，证明损失与程序违法之间的因果关系
- 建议聘请法律顾问评估具体违规情节，制定针对性整改方案
