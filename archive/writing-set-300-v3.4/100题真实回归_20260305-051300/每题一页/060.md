# 第 060 题

## 问题
单一来源采购的法律定义是什么？

## 结果状态
- status: OK
- evidence_count: 6
- top_score: 0.36581149405921237

## Top1 证据标题
中华人民共和国财政部令第74号《政府采购非招标采购方式管理办法》 第四条 （一）

## 回答
1. 结论

单一来源采购的法律定义是：采购人从某一特定供应商处采购货物、工程和服务的采购方式。

2. 核心要求清单

- 单一来源采购属于非招标采购方式的一种
- 采用前须在省级以上财政部门指定媒体上公示，公示期不少于5个工作日
- 须符合法定情形并经财政部门批准

3. 依据条款

- **条款**：第二条第四款；**来源法规**：《政府采购非招标采购方式管理办法》（中华人民共和国财政部令第74号）

4. 适用条件

根据相关法规，采用单一来源采购方式须符合法定情形，主要包括：
- 只能从唯一供应商处采购的
- 发生了不可预见的紧急情况不能从其他供应商处采购的
- 必须保证原有采购项目一致性或者服务配套的要求，需要继续从原供应商处添购，且添购资金总额不超过原合同采购金额10%的

5. 风险提示

- 单一来源采购因缺乏竞争，须严格审查是否符合法定情形
- 公示内容须包含采用原因说明、唯一供应商信息及专业人员论证意见
- 未经法定程序擅自采用单一来源采购方式，可能被认定为违法违规采购
