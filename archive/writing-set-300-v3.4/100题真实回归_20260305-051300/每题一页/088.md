# 第 088 题

## 问题
请解释重大违法记录与公开招标的适用边界。

## 结果状态
- status: OK
- evidence_count: 6
- top_score: 0.2699276590987238

## Top1 证据标题
中华人民共和国政府采购法  第二十二条 （一）

## 回答
**一、结论**

重大违法记录是供应商参加政府采购活动的**资格条件**，与采购方式（公开招标等）无直接对应关系。无论采用何种采购方式，供应商均须满足"前三年内无重大违法记录"的要求。采购方式的选择应依据采购金额、政策目标等因素确定。

**二、核心要求清单**

1. **时间范围**：供应商参加政府采购活动前**三年内**在经营活动中没有重大违法记录
2. **适用主体**：所有供应商（法人或其他组织、自然人）
3. **证明方式**：供应商须提供**书面声明**
4. **采购方式**：与公开招标、邀请招标、竞争性谈判等采购方式无直接关联

**三、依据条款**

- 条款：第二十二条第（五）项；来源法规：《中华人民共和国政府采购法》
- 条款：第二十二条第一款第（四）项；来源法规：《中华人民共和国政府采购法实施条例》
- 条款：第七条第（六）项；来源法规：《政府购买服务管理办法（暂行）》

**四、适用条件**

1. **时间起算**：从供应商报名参加政府采购活动之日起往前追溯三年
2. **违法类型**：须为经营活动中的重大违法记录，一般指受到较大数额罚款、责令停产停业、吊销许可证照等行政处罚
3. **采购项目有特殊要求**：采购人可规定供应商的特定条件，但不得以不合理条件实行差别待遇或歧视待遇

**五、风险提示**

1. 供应商须对书面声明的真实性负责，提供虚假声明将承担相应法律责任
2. 采购人应依法审查供应商资格条件，但不得超越法律法规设定额外门槛
3. 重大违法记录的认定标准目前法规未详细界定，实际执行中可能产生争议
