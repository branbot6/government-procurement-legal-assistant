# 第 008 题（基础概念与实务100）

## 问题
什么是政府采购限额标准？

## 回答
## 结论

政府采购限额标准是指由省级以上人民政府依法公布的，用于界定政府采购法适用范围的金额门槛。采购金额达到该标准以上的货物、工程和服务采购行为，须依照《政府采购法》相关规定执行。

## 核心要求清单

1. **门槛界定**：采购限额标准是划分政府采购与非政府采购的法定界限
2. **资金来源**：使用财政性资金
3. **适用情形**：采购集中采购目录以外的，且达到限额标准以上的货物、工程和服务
4. **标准制定**：由省级以上人民政府公布（国务院或省级财政部门具体制定）
5. **执行要求**：达到限额标准的项目须按政府采购法规执行

## 依据条款

- **条款**：第二条第二款
  > 本法所称政府采购，是指各级国家机关、事业单位和团体组织，使用财政性资金采购依法制定的**集中采购目录以内的或者采购限额标准以上的**货物、工程和服务的行为。
  
  > 政府集中采购目录和采购限额标准依照本法规定的权限制定。
  > **来源法规**：中华人民共和国政府采购法

- **条款**：第八条第二项
  > 省级以上人民政府公布的**集中采购目录、政府采购限额标准和公开招标数额标准**
  > **来源法规**：政府采购信息公告管理办法

## 适用条件

| 条件 | 说明 |
|------|------|
| 主体 | 各级国家机关、事业单位和团体组织 |
| 资金 | 使用财政性资金 |
| 对象 | 货物、工程和服务 |
| 金额标准 | 达到省级以上人民政府公布的采购限额标准 |

**地方示例**（江苏省2025年标准）：
- 货物、服务类：省本级100万元；其他市县50万元
- 工程类：省本级100万元；市县60万元

## 风险提示

- 未达到采购限额标准的采购，不强制适用政府采购法，但仍须遵守相关财务管理规定
- 达到限额标准而未按政府采购程序执行的，可能面临财政处理处罚
