# 第 026 题（基础概念与实务100）

## 问题
什么是询价采购？

## 回答
## 结论

询价采购是指询价小组向符合资格条件的供应商发出采购货物询价通知书，要求供应商一次报出不得更改的价格，采购人从询价小组提出的成交候选人中确定成交供应商的采购方式。属于《政府采购法》规定的非招标采购方式之一。

## 核心要求清单

1. **询价小组组成**：询价小组由采购人的代表和有关专家共三人以上的单数组成，其中专家的人数不得少于成员总数的三分之二
2. **供应商资格**：向符合资格条件的供应商发出询价通知书
3. **报价规则**：供应商一次报出不得更改的价格（不得更改）
4. **确定成交供应商**：采购人从询价小组提出的成交候选人中确定
5. **最低价成交原则**：通常选择报价最低的供应商成交
6. **竞争要求**：符合竞争要求的供应商不少于3家

## 依据条款

- **条款**：第二条；**来源法规**：政府采购非招标采购方式管理办法(中华人民共和国财政部令第74号)
- **条款**：第十二条；**来源法规**：中华人民共和国政府采购法
- **条款**：第五十条；**来源法规**：政府采购非招标采购方式管理办法(中华人民共和国财政部令第74号)
- **条款**：第三十三条；**来源法规**：中华人民共和国政府采购法实施条例（中华人民共和国国务院令第658号）
- **条款**：第四条；**来源法规**：扬州市本级政府采购环节操作规范实施细则

## 适用条件

1. 适用于货物采购，通常为规格、标准统一，现货货源充足且价格变化幅度小的项目
2. 采购金额未达到公开招标数额标准
3. 适用于采购人自行选择供应商的简单项目
4. 属于法定采购方式之一

## 风险提示

1. **终止情形**：出现以下情况应终止询价采购活动：不再符合询价采购适用情形；出现影响采购公正的违法违规行为；符合竞争要求的供应商或报价未超过采购预算的供应商不足3家
2. **人员回避**：询价小组组成人员与供应商有利害关系的必须回避
3. **保证金要求**：询价采购中要求供应商提交保证金的，参照投标保证金相关规定执行
4. **合规风险**：须按规定程序进行，避免因程序违规导致采购无效
