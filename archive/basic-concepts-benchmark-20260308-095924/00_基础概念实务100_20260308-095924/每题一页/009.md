# 第 009 题（基础概念与实务100）

## 问题
什么是政府购买服务？

## 回答
## 结论

政府购买服务是指通过发挥市场机制作用，把政府直接提供的一部分公共服务事项以及政府履职所需服务事项，按照一定的方式和程序，交由具备条件的社会力量和事业单位承担，并由政府根据合同约定向其支付费用的活动。

## 核心要求清单

1. **购买主体**：政府及其部门、参照公务员法管理的事业单位
2. **承接主体**：具备条件的社会力量和事业单位
3. **购买方式**：按照公开、公平、公正原则，通过公平竞争择优选择
4. **费用支付**：费随事转，政府根据合同约定向承接主体支付费用
5. **期限限制**：应严格限定在年度预算和中期财政规划期限内
6. **信息公开**：年度预算中政府购买服务总金额、项目信息需向社会公开
7. **禁止事项**：不得通过政府购买服务变相举债或违法违规融资

## 依据条款

1. **定义条款**：政府购买服务是指通过发挥市场机制作用，把政府直接提供的一部分公共服务事项以及政府履职所需服务事项，按照一定的方式和程序，交由具备条件的社会力量和事业单位承担，并由政府根据合同约定向其支付费用；购买服务范围应当根据政府职能性质确定，并与经济社会发展水平相适应；属于事务性管理服务的，应当引入竞争机制，通过政府购买服务方式提供。
   - 来源法规：《政府购买服务管理办法（暂行）》第二条

2. **基本原则**：积极稳妥、有序实施；科学安排、注重实效；公开择优、以事定费；改革创新、完善机制。
   - 来源法规：《政府购买服务管理办法（暂行）》第三条

3. **购买主体资格**：完全或主要承担行政职能的事业单位可以比照政府行政部门，作为政府购买服务的购买主体。
   - 来源法规：《关于做好事业单位政府购买服务改革工作的意见》第一条

4. **禁止纳入范围**：不属于政府职责范围的事项、应当由政府直接履职的事项、货物和工程及打包项目、融资行为、人员招聘用及劳务派遣等事项不得纳入政府购买服务范围。
   - 来源法规：《政府购买服务管理办法》第十条

5. **期限及预算要求**：政府购买服务期限应严格限定在年度预算和中期财政规划期限内；购买服务资金应当在年度预算和中期财政规划中安排。
   - 来源法规：《关于坚决制止地方以政府购买服务名义违法违规融资的通知》

6. **信息公开要求**：年度预算中政府购买服务总金额、纳入中期财政规划的购买服务总金额及项目有关预算信息，应按规定及时向社会公开。
   - 来源法规：《关于坚决制止地方以政府购买服务名义违法违规融资的通知》

## 适用条件

1. 适用于政府职能范围内的公共服务事项和履职所需服务事项
2. 适用于事务性管理服务，应当引入竞争机制
3. 购买服务应与改善民生密切相关、有利于转变政府职能
4. 承接主体需具备相应能力和条件，通过公平竞争方式确定

## 风险提示

1. **合规风险**：不得利用或虚构政府购买服务合同违法违规融资
2. **变相举债风险**：不得通过政府购买服务向金融机构、融资租赁公司等非金融机构进行融资，不得以任何方式虚构或超越权限签订应付（收）账款合同帮助企业融资
3. **期限超期风险**：购买服务期限不得超过中期财政规划期限
4. **信息公开不完整风险**：未按规定公开购买服务信息可能导致合规问题
