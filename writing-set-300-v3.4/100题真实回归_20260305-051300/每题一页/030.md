# 第 030 题

## 问题
根据《政府采购质疑和投诉办法》，第三十一条主要规定了什么？

## 结果状态
- status: OK
- evidence_count: 6
- top_score: 1.0

## Top1 证据标题
中华人民共和国财政部令第94号《政府采购质疑和投诉办法》 第三十一条 （一）

## 回答
## 结论

《政府采购质疑和投诉办法》第三十一条规定了财政部门对“投诉人就采购文件提起的投诉事项”的处理规则。

## 核心要求清单

1. **投诉查证属实** → 认定投诉事项成立
2. **不影响采购结果** → 继续开展采购活动
3. **影响或可能影响采购结果** → 按四类情形处理：
   - （一）未确定中标/成交供应商 → 责令重新开展采购活动
   - （二）已确定但未签合同 → 认定结果无效，责令重新开展采购活动
   - （三）合同已签但未履行 → 撤销合同，责令重新开展采购活动
   - （四）合同已履行 → 受害人可提起诉讼，由责任人承担赔偿责任

## 依据条款

- 条款：《政府采购质疑和投诉办法》第三十一条
- 来源法规：中华人民共和国财政部令第94号《政府采购质疑和投诉办法》

## 适用条件

- 投诉主体：投诉人
- 投诉对象：采购文件
- 适用情形：财政部门经查证投诉事项属实

## 风险提示

1. 投诉事项成立并不必然导致采购活动终止，需区分是否“影响或可能影响采购结果”
2. 合同已履行的，仅能通过民事诉讼途径救济损失，财政部门不再责令重新采购
3. 此条款仅适用于对采购文件的投诉，不适用于对采购过程或采购结果的投诉（后者适用第三十二条等条款）
