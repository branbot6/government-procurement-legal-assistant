# 第 058 题（合规整改）

## 问题
若历史项目在竞争性磋商中已形成程序性违法但合同已履行过半，如何在不扩大损失前提下做合规整改与责任切分？

## 结果状态
- status: OK

## Top1 证据标题
财政部关于印发《政府采购竞争性磋商采购方式管理暂行办法》的通知-财库〔2014〕214号 第三十四条 （一）

## 回答
## 1. 结论

对于竞争性磋商中已形成程序性违法但合同已履行过半的情形，应当立即停止违规行为、启动内部审查程序，并根据违法性质选择整改路径：若程序违法影响采购公正，应当终止项目重新开展采购；若程序违法可通过补正措施纠正，应当在控制损失前提下补正瑕疵，同时妥善处理已履行部分的合同关系。

## 2. 核心要求清单

- 立即停止可能扩大损失的继续履行行为
- 启动内部审查，认定程序违法的具体情形及严重程度
- 区分程序瑕疵与程序违法，判断是否影响采购公正
- 若影响采购公正的，依据《政府采购竞争性磋商采购方式管理暂行办法》第三十四条终止项目
- 若可补正的，在不扩大损失前提下采取补正措施
- 固定已履行合同的相关证据，合理确定后续处理方案
- 保留向财政部门报告或答复质疑的记录

## 3. 依据条款

- **条款**：第三十四条；**来源法规**：财政部关于印发《政府采购竞争性磋商采购方式管理暂行办法》的通知（财库〔2014〕214号）
- **条款**：第十四条；**来源法规**：中华人民共和国财政部令第94号《政府采购质疑和投诉办法》
- **条款**：第二条；**来源法规**：财政部关于印发《政府采购竞争性磋商采购方式管理暂行办法》的通知（财库〔2014〕214号）

## 4. 适用条件

**适用财库〔2014〕214号第三十四条终止程序的条件**：
- 出现影响采购公正的违法、违规行为
- 因情况变化，不再符合竞争性磋商采购方式适用情形

**适用补正措施的条件**：
- 程序瑕疵轻微且未影响采购公正
- 补正措施不会进一步扩大损失
- 供应商权益可通过其他方式获得救济

**责任切分的考量因素**：
- 违法行为的主动性与被动性
- 采购人与采购代理机构的各自职责范围
- 供应商是否存在过错
- 已履行部分的客观价值与付款情况

## 5. 风险提示

- 继续履行可能因程序违法被认定为无效或可撤销合同，反而扩大损失
- 若被供应商提出质疑或投诉，可能面临更大的法律风险
- 已支付款项的追回风险及违约责任承担问题
- 相关责任人员可能被追究行政责任
- 建议在做出决策前咨询财政部门或法律专业人士
