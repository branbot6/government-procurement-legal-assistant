# 第 088 题（程序与实体）

## 问题
评审专家回避问题中“程序违法”与“实体不当”应如何分别认定？对应救济路径是否相同？

## 结果状态
- status: OK
- evidence_count: 1
- top_score: 0.24455117540273075
- anchor: 程序违法

## Top1 证据标题
政府采购评审专家管理办法 第二十六条

## 回答
## 1. 结论

现有依据未对“程序违法”与“实体不当”作出明确区分，也未分别规定差异化的救济路径。基于《政府采购评审专家管理办法》第二十六条，回避问题本质上属于程序性要求，违反回避规定即为程序违法；而由此导致的评审结果不公平，属于实体后果。救济路径上，二者均通过质疑、投诉程序主张，财政部门可依法审查程序合法性及实体公正性。

## 2. 核心要求清单

- 评审专家与采购项目有利害关系的，应主动提出回避
- 财政部门、采购人或采购代理机构有权要求专家回避
- 有利害关系的情形包括：三年内曾在供应商处任职/担任顾问、配偶或直系亲属在供应商处任职/担任顾问、与供应商发生过法律纠纷、其他可能影响公正评标的情况

## 3. 依据条款

条款：第二十六条；来源法规：政府采购评审专家管理办法

## 4. 适用条件

回避制度适用于：评审专家与特定采购项目存在可能影响公正评审的利害关系。供应商认为专家未回避损害其合法权益的，可依法提出质疑、投诉。

## 5. 风险提示

**依据不足**：现有证据仅有一条，未涵盖“程序违法”与“实体不当”的法律定义及分别的救济途径。完整的认定标准和救济路径需结合《政府采购法》《政府采购法实施条例》《政府采购质疑和投诉办法》等综合判断。
