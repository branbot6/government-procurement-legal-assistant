# 第 110 题（程序与实体）

## 问题
质疑答复问题中“程序违法”与“实体不当”应如何分别认定？对应救济路径是否相同？

## 结果状态
- status: OK

## Top1 证据标题
中华人民共和国财政部令第94号《政府采购质疑和投诉办法》 第四条

## 回答
1) 结论

程序违法与实体不当的认定及救济路径存在区别。程序违法是指采购活动违反了法定程序性规定；实体不当是指采购文件、采购结果等实质内容存在违法违规。两者均可通过质疑投诉程序救济，但程序违法侧重于要求纠正违规程序，实体不当侧重于要求更正实质内容。

2) 核心要求清单

**程序违法的认定：**
- 采购方式选用违反规定
- 招标公告发布时间不足
- 评标程序不符合法定要求
- 未按规定组建评标委员会
- 评审标准未依法制定或未予公布

**实体不当的认定：**
- 采购文件设置不合理门槛
- 评审标准存在歧视性或排他性
- 中标结果与评审结果不一致
- 合同签订或变更违反法定要求

**救济路径：**
- 均需先向采购人或代理机构提出质疑
- 对质疑答复不满意可向财政部门投诉
- 财政部门依法对投诉事项进行审查处理

3) 依据条款

条款：第四条；来源法规：政府采购质疑和投诉办法（中华人民共和国财政部令第94号）

条款：第五条；来源法规：政府采购质疑和投诉办法（中华人民共和国财政部令第94号）

条款：第四十二条；来源法规：中华人民共和国政府采购法

条款：第六十六条；来源法规：中华人民共和国政府采购法实施条例

4) 适用条件

供应商认为采购活动存在程序违法或实体不当的，有权在知道或应知道其权益受到损害之日起7个工作日内提出质疑。采购人或代理机构应在收到质疑后7个工作日内作出答复。对答复不满意或采购人、代理机构未作答复的，可在收到答复或答复期满后15个工作日内向财政部门投诉。

5) 风险提示

程序违法与实体不当的界限有时存在交叉，财政部门在处理投诉时会综合判断。若投诉事项涉及专业判断问题，财政部门可依法通过专家评审等方式进行审查。供应商需注意救济时限，避免因逾期而丧失救济权利。
