# 第 032 题（反例推演）

## 问题
给一个质疑答复“形式合规但实质违规”的反例结构，再给对应的审查要点与纠偏动作。

## Top1 证据标题
中华人民共和国政府采购法  第四十二条 （一）

## 回答
**1) 结论**

质疑答复"形式合规但实质违规"的典型反例结构为：程序履行完整但答复内容实质性地违反了政府采购法律法规的实质性要求，如未依法依规核实质疑事项、答复理由与事实不符或规避法定责任。

**2) 核心要求清单**

- 质疑答复必须坚持"依法依规"原则（《政府采购质疑和投诉办法》第四条）
- 采购人负责供应商质疑答复，须对实质内容负责（《政府采购质疑和投诉办法》第五条）
- 采购文件（包括质疑答复）须妥善保存至少十五年（《政府采购法》第四十二条）
- 财政部门对质疑答复情况具有考核监督职责（《政府采购法实施条例》第六十六条）

**3) 依据条款**

- 条款：《政府采购质疑和投诉办法》第四条；来源法规：《中华人民共和国财政部令第94号‐政府采购质疑和投诉办法》
- 条款：《政府采购质疑和投诉办法》第五条；来源法规：《中华人民共和国财政部令第94号‐政府采购质疑和投诉办法》
- 条款：第四十二条；来源法规：《中华人民共和国政府采购法》
- 条款：第六十六条（一）至（四）；来源法规：《中华人民共和国政府采购法实施条例》
- 条款：第二十六条；来源法规：《中华人民共和国财政部令第74号‐政府采购非招标采购方式管理办法》

**4) 适用条件**

适用于采购人或采购代理机构作出质疑答复的任何政府采购项目。适用于财政部门对采购活动进行监督检查、考核及处理投诉时。

**5) 风险提示**

- **法律风险**：形式上走完程序但实质内容违规，可能被财政部门认定为违法并责令改正
- **监督风险**：财政部门对质疑答复情况纳入考核，发现实质违规将影响考核结果（《政府采购法实施条例》第六十六条）
- **救济风险**：供应商可能因"形式合规但实质违规"的答复继续提起投诉，导致采购活动陷入争议
- **档案风险**：质疑答复作为采购文件须保存十五年，实质违规的答复将成为不利证据
