# 第 036 题（合规整改）

## 问题
若历史项目在信用修复中已形成程序性违法但合同已履行过半，如何在不扩大损失前提下做合规整改与责任切分？

## Top1 证据标题
国务院办公厅关于印发《政府采购领域“整顿市场秩序、建设法规体系、促进产业发展”三年行动方案（2024—2026年）》的通知（国办发〔2024〕33号）

## 回答
1) 结论

在合同已履行过半的情况下，程序性违法且合同已履行的法律后果为：给采购人、供应商造成损失的，由责任人承担赔偿责任。合规整改应在不终止合同、不撤销合同的前提下，通过整改违法行为+责任切分+损失控制的方式处理。

2) 核心要求清单

- 立即停止程序性违法行为，继续履行合同避免损失扩大
- 主动向财政部门报告违法行为，争取从轻或减轻处罚
- 固定合同履行证据，证明合同已部分履行且质量合格
- 明确责任人范围，切分单位责任与个人责任
- 核算实际损失范围，避免承担扩大的损失
- 协商变更合同或签订补充协议，完善合规手续

3) 依据条款

- 条款：第七十三条第三项；来源法规：《中华人民共和国政府采购法》
- 条款：第七十二条第四款；来源法规：《中华人民共和国政府采购法实施条例》
- 条款：第三十一条第四项；来源法规：《中华人民共和国财政部令第94号-政府采购质疑和投诉办法》
- 条款：第三十二条第四项；来源法规：《中华人民共和国财政部令第94号-政府采购质疑和投诉办法》
- 条款：第五十五条第三项；来源法规：《中华人民共和国财政部令第74号-政府采购非招标采购方式管理办法》

4) 适用条件

- 适用于采购合同已经履行（全部或部分）的情形
- 适用于程序性违法（如信用修复程序违规、评审程序违法等）影响或可能影响中标、成交结果的情形
- 不适用于合同尚未签订或尚未履行的情形（那两种情形应撤销/终止合同）

5) 风险提示

- 程序性违法一旦被查实，合同有效性存在被质疑风险，但合同已履行的情况下不以撤销合同为原则
- 责任人需承担赔偿责任，范围限于实际损失而非预期利润损失
- 应主动整改+报告，争取认定为"经改正后仍然影响或者可能影响中标、成交结果"（实施条例第七十二条第二款），明确责任主体
- 合同履行过半意味着大部分义务已完成，此时终止合同损失更大，应优先采取补救措施而非撤销合同
