# 第 055 题（抽象原则落地）

## 问题
“公开透明、公平竞争、公正、诚实信用”在框架协议采购中的可审计落地动作分别是什么？如何形成可留痕证据链？

## 结果状态
- status: OK
- evidence_count: 6
- top_score: 0.47986426938522914
- anchor: 公开透明

## Top1 证据标题
中华人民共和国财政部令第87号《政府采购货物和服务招标投标管理办法》 第二十五条

## 回答
## 一、结论

“公开透明、公平竞争、公正、诚实信用”四项原则在框架协议采购中的可审计落地动作需贯穿采购全流程，通过制度设计、程序控制、信息留痕形成完整证据链。

---

## 二、核心要求清单

### （一）公开透明原则
| 落地动作 | 证据链留痕要求 |
|---------|--------------|
| 采购信息公告 | 在财政部门指定媒体发布采购公告，载明采购项目编号、名称、数量、技术要求 |
| 采购文件公示 | 采购文件在指定网站公示，供应商可自行下载 |
| 开标过程公开 | 开标全程公开进行，允许所有投标人参加，记录签到、密封检查、唱标等环节 |
| 评审标准公开 | 评标方法、评审标准在采购文件中载明并在评审前公开 |
| 结果公示 | 中标（成交）结果在指定媒体公示，公告期满足法定要求 |
| 合同公告 | 政府采购合同在签订后2个工作日内在指定媒体公告 |

### （二）公平竞争原则
| 落地动作 | 证据链留痕要求 |
|---------|--------------|
| 资格条件合规性审查 | 采购需求书不得含有以注册资本、资产总额等规模条件作为资格要求；不得将生产厂家授权等作为资格要求 |
| 需求合理性论证 | 采购需求需经市场调查或专家论证，技术、服务要求不得指向特定供应商或特定产品 |
| 评审因素量化 | 评审因素应细化和量化，与商务条件和采购需求对应，设置各区间对应分值 |
| 禁止歧视性条款 | 采购文件不得含有指向特定区域、行业业绩或特定专利、商标、品牌等条件 |

### （三）公正原则
| 落地动作 | 证据链留痕要求 |
|---------|--------------|
| 评审专家抽取 | 专家在省级政府采购专家库中随机抽取，抽取时间原则上在开标前半天或前一天 |
| 评审保密管理 | 评审专家名单在确定中标人前严格保密 |
| 回避制度执行 | 公布投标人名单，告知评审专家应当回避的情形 |
| 评审过程监督 | 监委会现场监督，评标全程录音录像 |
| 评分修正机制 | 仅限分值汇总错误、分项评分超范围、客观评审因素评分不一致、评分畸高畸低四种情形可修正 |

### （四）诚实信用原则
| 落地动作 | 证据链留痕要求 |
|---------|--------------|
| 低价合理性审查 | 评标委员会要求报价明显低于其他投标人的供应商提供书面说明，必要时提交证明材料 |
| 合同实质性履行 | 合同签订不得对招标文件确定事项和中标人投标文件作实质性修改 |
| 履约验收 | 按合同约定技术、服务、安全标准验收，验收报告签字存档 |
| 质疑投诉处理 | 供应商质疑依法依规答复，投诉事项基于质疑范围 |
| 档案保存 | 项目档案保存期限不少于15年，含全套采购文件及录音录像资料 |

## 三、依据条款

1. **条款**：招标文件、资格预审文件的内容不得违反公开透明、公平竞争、公正和诚实信用原则
   - **来源法规**：政府采购货物和服务招标投标管理办法（财政部令第87号）第二十五条

2. **条款**：政府采购活动遵循公开透明、公平竞争、公正和诚实信用原则
   - **来源法规**：扬州市本级政府采购环节操作规范实施细则（扬财规〔2018〕2号）第三条

3. **条款**：采购人提供的采购需求书内容应符合法律、行政法规、强制性标准、政府采购政策要求，不得规定以注册资本、资产总额等规模条件作为资格要求；不得将生产厂家授权作为资格要求；设定的条件与采购项目实际需要相适应；技术、服务等要求不得指向特定供应商、特定产品；不得以特定行政区域或特定行业业绩作为加分条件；不得限定特定专利、商标、品牌或供应商
   - **来源法规**：扬州市本级政府采购环节操作规范实施细则（扬财规〔2018〕2号）第六条第二款

4. **条款**：评审因素应当细化和量化，且与相应的商务条件和采购需求对应
   - **来源法规**：扬州市本级政府采购环节操作规范实施细则（扬财规〔2018〕2号）第七条

5. **条款**：采购人或者代理机构应当对开标、评标现场活动进行全程录音录像，音像资料作为采购文件一并存档
   - **来源法规**：扬州市本级政府采购环节操作规范实施细则（扬财规〔2018〕2号）第二十条第九款

6. **条款**：采购项目档案包括预算执行文件、前期准备文件、开标文件、评审文件、中标（成交）文件、合同文件及开评标现场录音录像资料等，档案保存期限为从采购结束之日起至少保存15年
   - **来源法规**：扬州市本级政府采购环节操作规范实施细则（扬财规〔2018〕2号）第四十二条

---

## 四、适用条件

适用于市级政府集中采购目录以内或市级政府集中采购目录以外、采购限额标准以上的货物、工程和服务采购项目。框架协议采购作为法定采购方式的一种，应遵循上述操作规范。

---

## 五、风险提示

1. **程序合规风险**：采购文件未按规定公示、公告期不足、评审专家未随机抽取等可能导致采购结果无效
2. **公平竞争风险**：资格条件设置不合理、评审因素指向特定供应商等易引发质疑投诉
3. **证据缺失风险**：未全程录音录像、档案资料不完整将导致审计追溯困难
4. **合同履行风险**：合同签订实质性修改招标文件内容、履约验收把关不严可能导致违约责任
