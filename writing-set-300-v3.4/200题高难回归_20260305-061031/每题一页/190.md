## 回答
## 询价采购法律定义、构成要件与排除要件

### 1. 结论

询价采购是政府采购非招标方式之一，指询价小组向符合资格条件的供应商发出询价通知书，要求供应商一次报出不得更改的价格，采购人从询价小组提出的成交候选人中确定成交供应商的采购方式。其核心特征为“一次报价、不得更改”，适用于规格标准统一、现货货源充足且价格变化幅度小的货物采购。

### 2. 核心要求清单

| 类型 | 要求内容 |
|------|----------|
| **主体要求** | 须成立询价小组，成员包括评审专家 |
| **供应商数量** | 邀请不少于3家符合资格条件的供应商 |
| **报价规则** | 供应商一次报出不得更改的价格 |
| **确定方式** | 采购人从询价小组提出的成交候选人中确定成交供应商 |
| **适用情形** | 货物规格标准统一、现货货源充足、价格变化幅度小 |

### 3. 依据条款

- **条款**：第二条
  **来源法规**：《政府采购非招标采购方式管理办法》（中华人民共和国财政部令第74号）

- **条款**：第十二条
  **来源法规**：《政府采购非招标采购方式管理办法》（中华人民共和国财政部令第74号）

- **条款**：第三十三条
  **来源法规**：《中华人民共和国政府采购法实施条例》（中华人民共和国国务院令第658号）

### 4. 适用条件

询价采购的适用需同时满足以下条件：
- 采购标的为货物（工程和服务一般不适用）
- 货物规格标准统一
- 现货货源充足
- 价格变化幅度小
- 采购金额未达到公开招标数额标准

### 5. 风险提示

**最容易被误用的口径**：在实际操作中，询价采购最常被误用于以下情形：

1. **适用范围扩大化**：将询价采购用于工程采购或服务采购，或者用于规格不统一、并非现货的货物采购。

2. **程序简化**：未按规定成立询价小组，或未向不少于3家供应商发出询价通知书。

3. **报价规则违反**：允许供应商进行多轮报价或价格谈判，违背“一次报出不得更改”的核心规则。

4. **资格条件审查不严**：未核实供应商是否具备相应资格条件。

根据《政府采购法》第三十二条，询价采购应适用于“采购的货物规格、标准统一、现货货源充足且价格变化幅度小的政府采购项目”。超出此范围适用询价采购属于违规行为。
