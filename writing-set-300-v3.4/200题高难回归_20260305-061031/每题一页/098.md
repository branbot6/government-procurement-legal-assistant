# 第 098 题（反例推演）

## 问题
给一个投诉处理“形式合规但实质违规”的反例结构，再给对应的审查要点与纠偏动作。

## Top1 证据标题
中华人民共和国财政部令第94号《政府采购质疑和投诉办法》 第三十五条

## 回答
1. 结论

投诉处理"形式合规但实质违规"的反例结构：财政部门在投诉处理过程中程序操作看似规范（如依法受理、按时答复、档案齐全），但实质上存在滥用职权、玩忽职守、徇私舞弊等违法违纪行为，导致处理结果损害供应商合法权益。

2. 核心要求清单

- 投诉处理须事实清楚、证据充分、定性准确
- 处理程序与处理结果须同时合规
- 财政部门须客观公正履行审查义务
- 评审活动须独立进行并承担法律责任
- 中小企业认定须依规进行

3. 依据条款

条款：财政部门及其工作人员在履行投诉处理职责中违反本办法规定及存在其他滥用职权、玩忽职守、徇私舞弊等违法违纪行为的，依照有关规定追究相应责任；涉嫌犯罪的，依法移送司法机关处理。；来源法规：政府采购质疑和投诉办法（财政部令第94号）第三十八条

条款：竞争性谈判小组或者询价小组成员应当履行下列义务：（一）遵纪守法，客观、公正、廉洁地履行职责；（二）根据采购文件的规定独立进行评审，对个人的评审意见承担法律责任；...；来源法规：中华人民共和国财政部令第74号《政府采购非招标采购方式管理办法》第九条

条款：政府采购监督检查、投诉处理及政府采购行政处罚中对中小企业的认定，由货物制造商或者工程、服务供应商注册登记所在地的县级以上人民政府中小企业主管部门负责。；来源法规：政府采购促进中小企业发展管理办法第十六条

4. 适用条件

- 财政部门已启动投诉处理程序并形成档案（形式合规）
- 但处理过程中存在事实认定错误、证据采信不当、法律适用错误等实质违规
- 或评审专家未独立客观评审导致结论错误
- 或中小企业认定程序违法导致认定结果错误

5. 风险提示

"形式合规但实质违规"的法律风险：

1. **行政责任风险**：财政部门及其工作人员可能因滥用职权、玩忽职守、徇私舞弊被追究行政责任（依据：财政部令第94号第三十八条）

2. **刑事责任风险**：涉嫌犯罪的，依法移送司法机关处理

3. **救济风险**：供应商可依法提起行政复议或行政诉讼，导致原处理决定被撤销

4. **监督风险**：财政部门须配合有关部门依法进行的监督检查（依据：财政部令第94号第三十五条），且投诉处理结果须依法公开（依据：财政部令第101号第三条）

审查要点：重点审查事实认定是否清楚、证据是否充分、法律适用是否正确、程序是否合法、结论是否公正。

纠偏动作：

1. 撤销或变更原处理决定
2. 重新调查核实并作出处理
3. 对相关责任人员追究责任
4. 完善内部审查机制，确保程序与实体并重
